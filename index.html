Pengaturan Sinkronisasi Google Form
Hubungkan dashboard dengan Google Form via Apps Script
Cara menghubungkan
  1. Buka Google Form Anda → klik ikon Sheets untuk membuat spreadsheet response
  2. Di Spreadsheet, buka Extensions → Apps Script
  3. Hapus kode default, paste kode dari file apps-script.gs yang sudah diunduh
  4. Klik Deploy → New deployment → Web App
  5. Set Execute as: Me dan Who has access: Anyone
  6. Klik Deploy, copy URL yang muncul
  7. Paste URL di bawah ini
Contoh: https://script.google.com/macros/s/AKfycby.../exec
Tandai Request Selesai
Request akan ditandai selesai.
